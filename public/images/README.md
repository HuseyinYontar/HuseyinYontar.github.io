# Placeholder Images

Place your images in this directory:

- `profile.jpg` - Your profile photo for the About page (recommended: 400x400px)
- `og-image.png` - Open Graph image for social shares (recommended: 1200x630px)

## Project Images

Create a `projects/` subdirectory for project images:
- `projects/portfolio.png`
- `projects/sample.png`
- etc.
